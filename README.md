# Partner Scorecard

Interactive dashboard for evaluating, scoring, and prioritizing creator and affiliate partners.

---

## How to Use

**Just click the link you were sent.** The app runs in your browser — nothing to install.

---

## What It Does

| Screen | Purpose |
|---|---|
| **Partner List** | Filter and sort all 25 partners by platform, niche, risk, and score |
| **Partner Detail** | Deep-dive into scores, stats, and AI-generated analysis |
| **Compare Partners** | Side-by-side radar charts and score comparisons |
| **Weekly Summary** | Top picks, risk flags, and suggested next actions |

---

## Running Locally (optional)

If you prefer to run it on your own machine:

**Mac:** Double-click `start.command`
**Windows:** Double-click `start.bat`

Requires [Python 3.9+](https://www.python.org/downloads/). First launch takes ~30 seconds to set up.
